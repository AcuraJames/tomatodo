# Ручное тестирование v0.1.15

## 1. Темы в селекте таймера
1. Переключи на **светлую тему** — селект задачи: белый фон, тёмный текст
2. Переключи на **тёмную тему** — селект задачи: тёмный фон (`#1a1a2e`), светлый текст (`#e2e8f0`)
3. Включи **Zen mode** — проверь что селект также корректно отображается в обеих темах
4. В попапе создания задачи — select проекта тоже должен корректно работать в обеих темах

## 2. Попап создания задачи в таймере
1. На странице таймера нажми `+` (новая задача)
2. Должен открыться **модальный попап** по центру экрана (как попап проекта)
3. Заголовок «Новая задача», крестик закрытия справа
4. Поле ввода «Название задачи...»
5. Выбор проекта (select)
6. Выбор срока (календарь)
7. Кнопки «Отмена» и «Создать»

## 3. Календарь — создание задач
### В таймере:
1. Открой попап создания задачи
2. Нажми на кнопку календаря — должен открыться попап календаря
3. Выбери дату — календарь закроется, дата отобразится как «ДД.ММ.ГГ»
4. Нажми крестик на дате — дата очистится
5. **Прошлые даты заблокированы** — кнопки серые, `cursor-not-allowed`
6. Кнопка «Назад» блокируется на текущем месяце

### В списках (Входящие, Планы, Проекты):
1. Рядом с инпутом создания задачи должен быть виден календарь
2. Выбери дату — задача автоматически попадёт в нужный список

### В списке «Сегодня»:
1. Календарь **не отображается** — задачи идут напрямую в «Сегодня»

## 4. Автоматическое определение списка по дате
1. Создай задачу с датой **сегодня** → попадёт в «Сегодня»
2. Создай задачу с датой **завтра** → попадёт в «Планы»
3. Создай задачу с датой **через 6 дней** → попадёт в «Планы»
4. Создай задачу с датой **через 7+ дней** → попадёт в «Планы»
5. Создай задачу **без даты** → попадёт во «Входящие» (или текущий список)
6. Просроченные задачи — дата **красная**

## 5. Break не убивается при сворачивании
1. Запусти фокус → дождись break
2. Сверни вкладку на 2-3 секунды
3. Вернись — таймер отдыха продолжает тикать

## 6. Восстановление таймера
1. Запусти фокус → подожди 10 сек → перезагрузи страницу
2. Таймер показывает оставшееся время и переключается в **паузу**

## 7. 404 маршрут
1. Перейди на `/asdfgh` → отображается страница «404 — Страница не найдена»
2. Кнопки «На главную» и «Назад» работают
